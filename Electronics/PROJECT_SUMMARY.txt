## File Descriptions

### Core Implementation

#### `json_scraper.py` (450+ lines)
**Purpose**: Web scraping and data extraction

**Key Classes**:
- `ElectronicsCategoryStructure` - Enum for category types
- `ElectronicsJsonScraper` - Main scraper class

**Key Methods**:
```python
# Category Management
- get_main_subcategories()      # Fetch all 17 electronics categories
- get_category_structure()       # Detect if catChilds/subcategories/direct

# Listing Data
- get_listings()                 # Fetch listings page
- get_listing_details()          # Fetch detail page
- download_image()               # Download product images

# Utilities
- format_relative_date()         # Format dates
- extract_attributes()           # Parse specifications
- get_page_json_data()           # Extract JSON from HTML
```

**Usage**:
```python
scraper = ElectronicsJsonScraper()
main_cats = await scraper.get_main_subcategories()
structure_type, children = await scraper.get_category_structure(slug)
listings, total_pages = await scraper.get_listings(category_slug, page=1)
```

#### `main.py` (470+ lines)
**Purpose**: Orchestration and workflow management

**Key Classes**:
- `ElectronicsScraperOrchestrator` - Main orchestrator

**Key Methods**:
```python
# Initialization
- initialize()                   # Set up scraper and S3
- cleanup()                      # Clean up resources

# Scraping Workflow
- scrape_all_main_categories()   # Process all 17 categories
- scrape_main_category()         # Process one category
- scrape_child_category()        # Process one child/subcategory

# Detail Fetching
- fetch_listing_details_batch()  # Get details for listings

# Data Export
- save_all_to_s3()               # Create Excel & upload to S3
```

**Workflow**:
1. Get all main categories (17 total)
2. For each category:
   - Detect structure (catChilds/subcategories/direct)
   - Process children/subcategories
   - Fetch all listings and details
3. Create Excel files organized by structure
4. Upload to S3 with date partitioning

#### `s3_helper.py` (390+ lines)
**Purpose**: AWS S3 integration and file management

**Key Class**:
- `S3Helper` - S3 client wrapper

**Key Methods**:
```python
# File Upload
- upload_file()                  # Upload Excel files
- upload_file_obj()              # Upload file objects
- upload_json_data()             # Upload JSON data
- upload_image()                 # Upload images

# File Management
- list_files()                   # List files in partition
- delete_file()                  # Delete files
- get_partition_prefix()         # Get S3 partition path

# URLs
- generate_s3_url()              # Create S3 URLs
- get_s3_url()                   # Get object URL
```

**Configuration**:
```python
AWS_PROFILE_NAME = "Your-Profile-Name"  # AWS SSO profile
AWS_REGION = "us-east-1"                # AWS region
```

## Execution Flow

### Single Run (python main.py)
```
1. Initialize (3-5 sec)
   ├─ Load configuration
   ├─ Connect to S3
   └─ Set up temp directory

2. Scrape Categories (2-4 hours)
   ├─ Fetch 17 main categories
   ├─ For each main category:
   │  ├─ Detect structure
   │  ├─ Process children/subcategories
   │  ├─ Fetch listings (all pages)
   │  ├─ Get listing details
   │  └─ Download images
   └─ Total: 12,000+ listings, 30,000+ images

3. Create Excel Files (5-10 min)
   ├─ Create workbooks (one per category)
   ├─ Add sheets (one per child)
   ├─ Add data
   └─ Total: 17 files

4. Upload to S3 (15-30 min)
   ├─ Upload Excel files
   ├─ Upload images
   └─ Upload JSON summary

5. Cleanup (1-2 sec)
   ├─ Remove temp files
   └─ Close connections

Total Time: 2-4 hours
```
