## 🏗️ Architecture

```
WantedCarsJsonScraper (json_scraper.py)
    ↓ (fetches JSON from HTML)
    ├── get_subcategories() → [wanted-american-cars, wanted-european-car, wanted-asian-cars]
    ├── get_listings(slug, page) → [listing1, listing2, ...]
    ├── get_listing_details(slug) → {detailed info}
    └── download_image(url) → bytes
    
S3Helper (s3_helper.py)
    ↓ (uploads to AWS)
    ├── upload_file(local_path, s3_filename)
    ├── upload_image(image_data, subcategory_slug)
    └── generate_s3_url(s3_key)
    
WantedCarsScraperOrchestrator (main.py)
    ↓ (orchestrates everything)
    ├── scrape_all_subcategories()
    ├── fetch_listing_details_batch()
    └── save_all_to_s3()
```

## 🎓 How It Works

1. **Discovery Phase**
   - Fetch main wanted-cars page
   - Extract 3 subcategories from catChilds

2. **Listing Scraping Phase**
   - For each subcategory:
     - Fetch listings with pagination
     - Extract basic info (title, price, phone, etc.)

3. **Detail Fetching Phase**
   - For each listing:
     - Fetch detailed information page
     - Extract attributes and additional data
     - Download all images

4. **Upload Phase**
   - Create Excel file with sheets per subcategory
   - Upload Excel to S3
   - Upload all images to S3 (organized by subcategory)
   - Create and upload JSON summary

5. **Cleanup Phase**
   - Remove temporary files
   - Close S3 connection
   - Display summary statistics
